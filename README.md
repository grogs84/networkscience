# Network Science Book

Welcome to the Network Science book! This book is built using Jupyter Book.

## About This Book

This is an introductory placeholder for a Network Science book. The content is organized into chapters that can be found in the `content/` directory.

## Getting Started

To build this book locally:

1. Install the requirements:
   ```bash
   pip install -r requirements.txt
   ```

2. Build the book:
   ```bash
   jupyter-book build .
   ```

3. View the book by opening `_build/html/index.html` in your browser.

## Contents

This book covers various topics in Network Science, including:

- Introduction to networks and graph theory
- Network properties and metrics
- Community detection
- Network dynamics
- Applications of network science
